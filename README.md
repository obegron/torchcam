# TorchCam

Fullscreen back-camera preview with the torch kept on.

The app requests camera permission, opens the rear camera, enables torch mode in the preview request, and supports tap-to-focus. The close icon in the top-right exits the app.

## Build

```sh
./gradlew build
```

Debug APK:

```text
app/build/outputs/apk/debug/app-debug.apk
```
